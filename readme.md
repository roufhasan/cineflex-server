# CineFlex Server

This is a server project built on ExpressJS and MongoDB to handle [CineFlex](https://cineflex1.netlify.app/) project watchlisted item.

### Features

- Users can Add a movie or tv-show one time into the Database.
- Users can Delete a watchlisted movie or tv-show.
- Users can Delete All watchlisted items at one time.
